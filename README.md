# Game Dev Idle

A browser-based idle MMORPG inspired by FarmRPG.

Players become indie game developers.

Gameplay revolves around:

- Programming
- Designing games
- Collecting assets
- Fixing bugs
- Shipping games
- Completing contracts
- Building studios
- Researching technologies

Goals:

- Mobile-first
- Mostly text-based
- Thousands of items
- Long-term progression
- Community driven

## Development

The Foundation scaffold includes a Next.js web app, NestJS API, BullMQ worker, PostgreSQL, Redis, shared contracts, and Docker Compose.

See [local setup instructions](docs/SETUP.md).
